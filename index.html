Sambung ke database dulu

Buka fail index.html, cari bahagian KONFIGURASI di atas <script>, dan masukkan URL + anon key projek Supabase awak:

const SUPABASE_URL  = "https://xxxx.supabase.co";
const SUPABASE_ANON = "eyJhbGci...";

Cara dapatkan: Supabase → Project Settings → API. Penuh langkah dalam PANDUAN.md.
